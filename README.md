# ember-use-sound

_This is a [V2-format Addon](https://github.com/emberjs/rfcs/pull/507) with V1 compatibility_

## Compatibility

- ember-source v3.25+
- typeScript v4.2+
- ember-auto-import v2+

## Installation

```bash
npm install ember-use-sound
# or
yarn add ember-use-sound
# or
ember install ember-use-sound
```
